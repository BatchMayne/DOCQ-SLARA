ALSO YOU NEED TO UNDERSTAND DOCQ LOGIC


wrok - Wrok is a command where it connects to the files of DOCQ, mostly used in installations



set - Set is a command where you setup the Wrok command



inl - Inl means Install, used to install packages




akas - Akas is a Terminal updater, it is used to update the terminal when a package is installed



lkernel - lkernel is the DOCQ itself, mostly used to Delete cache or Files




docq.driver - Docq.driver is a function to access DOCQ's drivers, mostly used to install driver, manage driver, and delete driver (Deleting drivers will make DOCQ not work"
